Grid Grid Container: The element on which display: grid is applied. It’s the direct parent of all the grid items. In this example container is the grid container.
1) Grid Item: The children (i.e. direct descendants) of the grid container.

Sub-Item: Here the item elements are grid items, but sub-item isn’t.

2) Grid Line: The dividing lines that make up the structure of the grid. They can be either vertical (“column grid lines”) or horizontal (“row grid lines”) and reside on either side of a row or column. Here the yellow line is an example of a column grid line.
3) rid Cell: The space between two adjacent row and two adjacent column grid lines. It’s a single “unit” of the grid. Here’s the grid cell between row grid lines 1 and 2, and column grid lines 2 and 3.
4) Grid Track: The space between two adjacent grid lines. You can think of them as the columns or rows of the grid. Here’s the grid track between the second and third-row grid lines.
5) Grid Area: The total space surrounded by four grid lines. A grid area may be composed of any number of grid cells. Here’s the grid area between row grid lines 1 and 3, and column grid lines 1 and 3.
